# 商业分析框架速查

## 概述

本文档汇集企业研究中常用的 49 个商业分析框架，按七大类别组织索引。每个框架包含定义、适用场景和核心要点。部分框架在其他专题文档中有详细展开，本文以交叉引用方式标注，避免重复。

## 战略规划

### PESTEL 分析

政治、经济、社会、技术、环境、法律六维度宏观环境扫描。详见 [macro-environment.md](macro-environment.md)

### SWOT 分析

优势、劣势、机会、威胁四维度综合态势评估，含 SO/WO/ST/WT 四种组合策略。详见 [strategy-capability.md](strategy-capability.md)

### 波特五力模型

从新进入者、替代品、供应商、买方、现有竞争五种力量分析行业竞争结构和盈利潜力。详见 [industry-competition.md](industry-competition.md)

### 波士顿矩阵（BCG 矩阵）

市场增长率与相对市场份额二维度评估业务组合，分为明星、现金牛、问题、瘦狗四类。详见 [strategy-capability.md](strategy-capability.md)

### GE 矩阵

行业吸引力与业务竞争力多因素九象限分析，比 BCG 矩阵考虑更多变量。详见 [strategy-capability.md](strategy-capability.md)

### 安索夫矩阵

从产品和市场两个维度提供市场渗透、市场开发、产品开发、多元化四种增长路径。详见 [strategy-capability.md](strategy-capability.md)

### 麦肯锡 7S 模型

战略、结构、制度、风格、人员、技能、共同价值观七要素组织诊断。详见 [strategy-capability.md](strategy-capability.md)

### 3C 战略三角

公司、顾客、竞争对手三角营销战略分析。详见 [strategy-capability.md](strategy-capability.md)

### 价值链分析

将企业活动分为基本活动与支持活动，识别竞争优势来源。详见 [strategy-capability.md](strategy-capability.md)

### 核心竞争力分析（VRIO）

从价值性、稀缺性、难以模仿性、组织性四维度评估资源和能力是否构成持续竞争优势。详见 [strategy-capability.md](strategy-capability.md)

### 商业模式画布（Business Model Canvas）

亚历山大·奥斯特瓦德提出的一页纸商业模式描述工具，用 9 个构成要素系统呈现企业如何创造、传递和获取价值。

**适用场景**：创业项目商业模式设计、现有业务模式梳理与创新、商业计划书撰写前的结构化思考。

**9 个构成要素**：
1. **客户细分（CS）**——企业服务的目标客户群体
2. **价值主张（VP）**——为客户解决什么问题、满足什么需求
3. **渠道通路（CH）**——如何触达客户并交付价值
4. **客户关系（CR）**——与客户建立和维护什么类型的关系
5. **收入来源（RS）**——通过什么方式从客户获得收入
6. **核心资源（KR）**——支撑商业模式运转的关键资产
7. **关键业务（KA）**——企业必须执行的核心活动
8. **重要合作（KP）**——支撑商业模式的外部合作网络
9. **成本结构（CS）**——运营商业模式产生的主要成本

### 蓝海战略 / 战略画布

金伟灿和勒妮·莫博涅提出的价值创新理论，主张跳出现有竞争（红海），通过同时追求差异化和低成本开创无人竞争的新市场空间（蓝海）。

**适用场景**：行业竞争白热化、利润空间被压缩时寻找新增长路径；评估是否存在尚未被满足的市场需求。

**核心工具与步骤**：
1. **战略画布**——横轴列出行业竞争要素，纵轴为各要素的投入水平，绘制自身与竞争对手的价值曲线，识别趋同竞争模式
2. **四项行动框架（ERRC）**——剔除（Eliminate）行业中哪些被视为理所当然的要素？减少（Reduce）哪些要素可以大幅降低？增加（Raise）哪些要素应大幅提升？创造（Create）哪些行业从未提供的要素？
3. **重建市场边界的六条路径**——跨越替代性行业、跨越战略集团、跨越买方链、跨越互补性产品/服务、跨越功能与情感导向、跨越时间

详见 [strategy-capability.md](strategy-capability.md) 中蓝海战略章节的展开论述。

### 精益画布（Lean Canvas）

Ash Maurya 在商业模式画布基础上改良的创业专用工具，更聚焦问题验证和快速迭代。

**适用场景**：早期创业项目验证商业假设、MVP 规划、精益创业方法论实践。

**与商业模式画布的区别**：
1. 用**问题**替代"重要合作"——明确客户最痛的前三个问题
2. 用**解决方案**替代"关键业务"——针对问题提出的解决办法
3. 用**关键指标**替代"客户关系"——衡量业务健康度的核心数据
4. 用**独特优势**替代"核心资源"——竞争对手无法轻易复制的差异化因素
5. 保留客户细分、渠道、收入来源、成本结构四个要素不变

## 营销管理

### STP 分析

市场细分、目标市场选择、市场定位三步营销战略。详见 [consumer-insight.md](consumer-insight.md)

### 4P 营销组合

产品、价格、渠道、促销四要素营销组合。详见 [consumer-insight.md](consumer-insight.md)

### 4C 营销理论

消费者需求、成本、便利性、沟通四要素，从消费者视角重构营销组合。详见 [consumer-insight.md](consumer-insight.md)

### 4R 营销理论

关联、反应、关系、回报四要素，从关系视角重构营销组合。详见 [consumer-insight.md](consumer-insight.md)

### 4V 营销理论

差异化、功能弹性、附加价值、情感共鸣四要素的营销框架，强调在产品同质化时代通过非物质层面建立竞争优势。

**适用场景**：产品功能趋同的成熟市场、需要从价格竞争转向价值竞争时。

**核心维度**：
1. **差异化（Variation）**——产品、服务、品牌形象的独特性
2. **功能弹性（Versatility）**——产品功能的可定制性和灵活性
3. **附加价值（Value）**——超越产品本身的额外价值（售后、培训、社区等）
4. **情感共鸣（Vibration）**——品牌与消费者之间的情感连接和共鸣

### 产品生命周期

产品从导入到衰退的四阶段管理框架，不同阶段需要不同的竞争策略和资源配置。详见 [industry-competition.md](industry-competition.md) 中行业生命周期分析章节。

### 品牌资产模型

从知名度、联想、形象、忠诚度四个维度评估品牌价值。详见 [brand-value.md](brand-value.md)

### 顾客价值分析

系统识别客户需求层次并评估企业价值创造能力的方法。

**适用场景**：新产品开发前的需求挖掘、现有产品的价值提升、竞争差异化策略制定。

**核心步骤**：
1. **识别价值要素**——通过调研确定客户看重的功能、情感、社会价值要素
2. **评估重要性排序**——量化各要素对购买决策的影响权重
3. **竞争对比**——评估自身与竞争对手在各要素上的表现差异
4. **确定价值差距**——找出客户期望与实际感知之间的差距
5. **制定改进方案**——优先强化高重要性、低表现的价值要素

### 顾客满意度分析

收集客户反馈、评估满意水平、挖掘不满因素、制定改进措施的系统方法。详见 [loyalty-satisfaction.md](loyalty-satisfaction.md) 中 Satisfactor 模型和钱包分配法则。

### 客户旅程地图（Customer Journey Map）

以时间轴方式可视化客户与企业互动全过程的分析工具，识别关键触点和体验痛点。

**适用场景**：客户体验优化、服务流程改造、新产品/服务设计、跨部门协作对齐客户视角。

**核心步骤**：
1. **阶段划分**——将客户旅程分为认知、考虑、购买、使用、售后、推荐等阶段
2. **触点识别**——列出每个阶段客户与企业接触的所有渠道和方式
3. **情感曲线**——标记客户在每个触点的情感体验（满意/中性/痛苦）
4. **痛点与爽点标记**——识别体验断裂点和超出预期的亮点
5. **优化机会**——针对关键痛点设计改进方案，放大爽点创造差异化

详见 [consumer-insight.md](consumer-insight.md) 中客户旅程地图章节的展开论述。

## 竞争分析

### 竞争态势矩阵（CPM）

量化比较企业与竞争对手在关键成功因素上的表现。详见 [industry-competition.md](industry-competition.md) 和 [strategy-matrices.md](strategy-matrices.md)

### 市场吸引力矩阵

综合评估特定市场是否值得进入或继续投入的分析工具。

**适用场景**：新市场进入决策、现有市场组合优化、资源分配优先级排序。

**核心评估维度**：
1. **市场规模**——当前市场容量和可触达的市场空间
2. **增长率**——历史增长趋势和未来增长预测
3. **竞争强度**——现有竞争者数量、集中度和竞争激烈程度
4. **盈利水平**——行业平均毛利率和利润率
5. **进入壁垒**——技术、资本、渠道等进入门槛

### 关键成功因素分析（KSF）

识别在特定行业中决定企业成败的最重要因素。详见 [industry-competition.md](industry-competition.md)

### 资源基础观（RBV）

从企业内部资源和能力角度分析竞争优势来源的理论框架，认为异质性资源是超额利润的根本原因。

**适用场景**：评估企业核心竞争力的可持续性、并购标的的资源价值评估、内部能力建设规划。

**核心逻辑**：
1. **资源异质性**——企业拥有的资源和能力存在差异
2. **资源不可流动性**——某些资源无法在企业间自由转移
3. **VRIO 检验**——用价值性、稀缺性、难以模仿性、组织性四个标准逐一检验资源
4. **竞争优势分级**——竞争劣势 → 竞争均势 → 暂时竞争优势 → 持续竞争优势

### 动态能力理论

评估企业在快速变化环境中感知机会、整合资源和持续变革能力的理论框架。

**适用场景**：技术快速迭代的行业分析、评估企业转型和适应能力、创新管理评估。

**核心维度**：
1. **感知能力（Sensing）**——识别和评估外部环境中的机会与威胁
2. **抓取能力（Seizing）**——动员资源抓住机会、应对威胁
3. **转型能力（Transforming）**——重新配置组织资产和结构以保持竞争力

## 增长创新

### 波特通用战略

迈克尔·波特提出的三种基本竞争战略，企业必须在其中做出明确选择，避免"夹在中间"。

**适用场景**：企业竞争战略定位、业务单元战略选择。

**三种战略**：
1. **成本领先**——通过规模经济、流程优化、供应链管理实现行业最低成本，以低价或同价更高利润参与竞争
2. **差异化**——在产品、服务、品牌、技术等方面建立独特性，使客户愿意支付溢价
3. **集中化（聚焦）**——选择特定细分市场深耕，在该细分市场内执行成本领先或差异化战略

### 平台战略

构建连接多方参与者的平台生态，通过网络效应创造价值的商业模式。

**适用场景**：评估平台型企业的竞争优势和护城河、判断是否适合采用平台模式。

**核心要素**：
1. **多边市场**——连接两个或多个相互依赖的用户群体
2. **网络效应**——用户数量增加使平台价值对所有参与者提升（直接/间接网络效应）
3. **平台治理**——准入规则、质量控制、利益分配机制
4. **冷启动问题**——如何解决平台初期供需两端的引导难题

### 生态系统战略

与供应商、合作伙伴、互补品提供商、客户共同构建协同创造价值的商业生态。

**适用场景**：评估企业在产业生态中的位置和影响力、设计合作伙伴策略。

**核心维度**：
1. **生态位定位**——企业在生态系统中扮演什么角色（核心企业/互补者/供应商）
2. **价值共创机制**——参与者如何协同创造单独无法实现的价值
3. **协调与治理**——生态系统的规则、标准和激励机制
4. **生态健康度**——生产力、稳健性、生态位创造力三个评估指标

### 飞轮效应（Flywheel）

吉姆·柯林斯提出的概念，描述一系列相互强化的战略行动如何形成自我加速的良性循环，推动业务持续增长。

**适用场景**：识别企业增长的核心驱动循环、评估商业模式的自我强化特性、战略聚焦与资源分配。

**核心步骤**：
1. **识别飞轮要素**——找出业务中相互促进的关键环节（如亚马逊：更多选品 → 更好体验 → 更多流量 → 更多卖家 → 更多选品）
2. **确定驱动力**——找到启动飞轮的第一推动力
3. **消除阻力**——识别并移除阻碍飞轮转动的摩擦点
4. **持续加速**——通过持续投入核心环节使飞轮越转越快，形成竞争壁垒

## 组织执行

### 平衡计分卡（BSC）

从财务、客户、内部流程、学习与成长四个维度衡量企业绩效，确保短期行动与长期战略一致。详见 [thinking-models.md](thinking-models.md)

### 六西格玛

定义、测量、分析、改进、控制（DMAIC）五阶段质量管理方法论。详见 [thinking-models.md](thinking-models.md)

### PDCA 循环

计划、执行、检查、改进的持续改善循环。详见 [thinking-models.md](thinking-models.md)

### OKR 目标管理

英特尔创立、Google 推广的目标与关键结果管理框架，通过设定有挑战性的目标和可衡量的关键结果驱动组织对齐和聚焦。

**适用场景**：企业战略目标分解与落地、团队目标对齐、激发挑战性创新目标。

**核心要素**：
1. **目标（Objective）**——定性描述要达成的方向，应具有激励性和挑战性
2. **关键结果（Key Results）**——量化衡量目标达成度的 3-5 个指标，必须可度量
3. **对齐机制**——自上而下与自下而上相结合，确保各层级目标逻辑一致
4. **节奏与复盘**——通常按季度设定，定期检查进度，周期结束时评分复盘（0.6-0.7 分为理想区间，说明目标有足够挑战性）
5. **与 KPI 的区别**——OKR 强调挑战性和方向引领，允许不完全达成；KPI 强调底线达标和绩效考核

### 战略实施与执行

将战略意图转化为可操作行动并确保落地执行的管理流程。

**适用场景**：战略规划制定后的落地阶段、组织变革推进。

**核心步骤**：
1. **目标分解**——将战略目标拆解为部门和个人层面的可执行任务
2. **行动计划制定**——明确每项任务的时间表、责任人和交付物
3. **资源配置**——根据战略优先级分配预算、人力和技术资源
4. **过程监控**——建立里程碑和检查点，定期跟踪进度
5. **动态调整**——根据执行反馈和环境变化及时修正方向

### 风险管理框架

系统化识别、评估和应对企业面临的各类风险的管理方法。

**适用场景**：新业务进入前的风险评估、企业年度风险审查、重大决策前的风险分析。

**核心步骤**：
1. **风险识别**——全面扫描战略、运营、财务、合规等各类风险
2. **风险评估**——按发生概率和影响程度对风险进行优先级排序
3. **应对策略**——规避、转移、减轻或接受，为每类风险选择合适的应对方式
4. **监控与预警**——建立关键风险指标的持续监控和预警机制

## 决策思维

### 六顶思考帽

白色事实、红色情感、黑色风险、黄色积极、绿色创意、蓝色控制，六种平行思维方式。详见 [thinking-models.md](thinking-models.md)

### KT 决策法

状况分析、问题分析、决策分析、潜在问题分析四步系统决策方法。详见 [thinking-models.md](thinking-models.md)

### 5W2H 分析法

What、Why、Who、When、Where、How、How much 七维度系统化问题分析。详见 [thinking-models.md](thinking-models.md)

### 德尔菲法

多轮匿名专家意见征询，逐步收敛达成共识的预测方法。详见 [thinking-models.md](thinking-models.md)

### 决策树分析

树状图呈现决策路径和概率，辅助在不确定条件下做出最优选择。详见 [thinking-models.md](thinking-models.md)

### 头脑风暴法

延迟判断、追求数量、自由联想的团队创意激发方法。详见 [thinking-models.md](thinking-models.md)

## 财务分析

### 杜邦分析法

将 ROE 分解为利润率 × 资产周转率 × 财务杠杆，系统评价企业盈利能力。详见 [thinking-models.md](thinking-models.md)

### 比率分析法

通过计算和比较财务比率，多维度评估企业财务健康状况的方法。

**适用场景**：企业财务健康度评估、同行业企业横向比较、财务趋势分析。

**核心维度**：
1. **流动性比率**——流动比率、速动比率，衡量短期偿债能力
2. **盈利性比率**——毛利率、净利率、ROA、ROE，衡量盈利能力
3. **偿债能力比率**——资产负债率、利息保障倍数，衡量长期偿债能力
4. **运营效率比率**——存货周转率、应收账款周转率、资产周转率，衡量资产使用效率

### 本量利分析（CVP）

分析成本、业务量和利润之间关系，确定盈亏平衡点的决策工具。详见 [thinking-models.md](thinking-models.md)

### ABC 成本法（作业成本法）

按活动归集成本的精细化成本管理方法，解决传统成本分摊不准确的问题。

**适用场景**：产品线盈利能力精确评估、间接成本占比高的企业成本管理、定价决策支持。

**核心步骤**：
1. **识别作业活动**——将企业运营分解为具体的作业活动（如订单处理、质检、物流配送）
2. **归集作业成本**——将资源成本归集到各作业活动
3. **确定成本动因**——找出驱动每项作业成本的关键因素（如订单数、检验次数）
4. **分配到成本对象**——按成本动因将作业成本分配到产品、服务或客户

### 客户终身价值（CLV/LTV）

衡量单个客户在整个关系期间为企业贡献的总净利润的指标和分析方法。

**适用场景**：客户获取成本（CAC）合理性评估、高价值客户识别与资源分配、营销投入回报评估。

**核心要素**：
1. **基本公式**——CLV = 平均客户价值 × 平均客户寿命；或 CLV = (平均订单金额 × 购买频率 × 客户寿命) - 获客成本
2. **关键影响因子**——客单价、购买频率、客户留存率、毛利率、贴现率
3. **CLV/CAC 比值**——健康企业通常 CLV:CAC >= 3:1
4. **分群应用**——按客户群体分别计算 CLV，指导差异化运营策略

详见 [loyalty-satisfaction.md](loyalty-satisfaction.md) 中客户终身价值章节的展开论述。

## 选用指南

面对具体研究任务时，应根据问题性质选择合适的框架组合：

- **宏观层面**：PESTEL + 五力模型，理解外部环境和行业结构
- **企业层面**：SWOT + 价值链 + 7S，评估内部能力和战略匹配度
- **业务层面**：BCG/GE 矩阵 + 安索夫矩阵，优化业务组合和增长方向
- **商业模式层面**：商业模式画布 + 精益画布，设计和验证商业逻辑
- **消费者层面**：STP + 4P/4C + 客户旅程地图 + 价值要素金字塔
- **客户关系层面**：WAO + Satisfactor + CLV/LTV，详见 [loyalty-satisfaction.md](loyalty-satisfaction.md)
- **财务层面**：杜邦分析 + 比率分析 + CVP，评估财务健康度

框架是思考的脚手架而非思考的替代品。真正的洞察来自于对行业和企业的深刻理解，框架只是帮助组织和呈现这些理解的工具。
